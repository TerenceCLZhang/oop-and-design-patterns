# SOLID Principles

Used to create software that is maintainable, scalable, flexible, and robust, which leads to faster development, easier bug fixing, and simpler integration of new features over time.

S: Single Responsibility Principle (SRP)
O: Open-closed Principle (OCP)
L: Liskov Substitution Principle (LSP)
I: Interface Segregation Principle (ISP)
D: Dependency Inversion Principle (DIP)

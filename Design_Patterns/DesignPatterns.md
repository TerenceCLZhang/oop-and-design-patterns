# Design Patterns

From the book "Design Patterns: Elements of Reusable Object-Oriented Software" by the "Gang of Four" (Erich Gamma, Richard Helm, Ralph Johnson, John Vlissides).

There are three main group of design patterns with 23 individual design patterns.

- **Creational Patterns**: Focus on object creation methods to improve flexibility and reuse.
- **Structural Patterns**: Focus on class and object composition to form larger structures.
- **Behavioural Patterns**: Focus on interaction and responsibility assignment among objects.
